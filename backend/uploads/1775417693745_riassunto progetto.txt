# 🧠 **RIASSUNTO PROGETTO (versione corta)**

## 🎯 Obiettivo

Sviluppare un **MVP web app** per il **trasferimento sicuro di file** (DataShare), pronto in 4 settimane per investitori.

---

## ⚙️ Stack

* Backend: **Spring Boot (Java)**
* Frontend: **Angular**
* Database: **PostgreSQL**
* Storage: **File system locale**

---

## 🔑 Funzionalità MVP

* Signup / Login (JWT)
* Upload file
* Generazione link download (token)
* Download via link
* Historique files
* Delete file

---

## 🗄 Modello dati

* **User** (id, email, password)
* **File** (id, filename, path, token, expiration, user_id)

👉 Relazione: 1 User → N Files

---

## 🔄 Flusso principale

1. Login → JWT
2. Upload → file salvato + token
3. Link → download file
4. User → vede storico + delete

---

## 🔐 Sicurezza

* Password hash (bcrypt)
* JWT auth
* Token download non predicibile
